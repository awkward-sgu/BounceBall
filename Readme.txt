최종프로젝트: BounceBall

모바일 게임인 BounceBall을 약간 형태를 바꿔 파쿠르 게임 형식으로 만들어 봤습니다!



마우스 클릭과 좌우방향키로만 동작합니다.

메뉴 창에는 플레이, 튜토리얼, 랜덤, 나가기가 있는데 먼저 튜토리얼을 한 판 한 뒤에 메인 게임(플레이)을 하시는 것을 추천드립니다.
메인 게임은 총 7단계로 구성되어 있으며, 1단계부터 순차적으로 난이도가 올라갑니다.
오른쪽에 버튼이 2개 있을 건데, >> 버튼은 다음 단계로 넘어가는 버튼이고 X 버튼은 게임 종료하고 메뉴로 돌아가는 버튼입니다.

랜덤은 맵이 랜덤으로 생성되는 모드인데, 메인 게임을 한 뒤에 해보시면 좋습니다. 난이도는 Easy, Intermediate, Hard가 있으며 각 단계에 맞게 맵이 생성됩니다.
랜덤에서는 맵을 클리어하면 또다른 랜덤 맵이 계속해서 생성됩니다.
