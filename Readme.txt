최종프로젝트: BounceBall

모바일 게임인 BounceBall을 약간 바꿔 파쿠르 게임 형식으로 만들어 봤습니다!



마우스 좌클릭 (버튼)과 좌우방향키로 동작합니다. 공은 계속해서 자동으로 위아래로 통통 튀며, 플레이어는 좌우로만 이동할 수 있습니다.
모든 UI는 영어입니다.
오른쪽 상단에 전체화면 버튼이 있습니다.


처음 보이는 메뉴 창에는 플레이, 튜토리얼, 모드, 나가기가 있습니다. 먼저 튜토리얼을 한 판 하신 뒤에 메인 게임(플레이)을 하시는 것을 추천드립니다.
튜토리얼은 플레이 방법을 처음부터 알려주는 방식으로 제작하였습니다.


메인 게임은 난이도 easy, medium, hard 각 10단계로 구성되어 있으며, 1단계부터 순차적으로 난이도가 올라갑니다.
오른쪽에 버튼이 2개가 있습니다. >> 버튼은 다음 단계로 넘어가는 버튼이고 X 버튼은 게임을 종료하고 메뉴로 돌아가는 버튼입니다.




모드에는 랜덤과 메테오가 있습니다.

랜덤은 맵이 랜덤으로 생성되는 모드인데 난이도는 메인 게임과 똑같이 easy, medium, hard가 있으며 각 난이도에 맞게 맵이 생성됩니다.
메인 게임에서 플레이한 패턴을 랜덤하게 생성하기 때문에  메인 게임을 플레이하신 뒤에 해보시는 게 좋습니다.
랜덤에서는 맵을 클리어하면 또다른 랜덤 맵이 계속해서 생성됩니다.

메테오는 맵이 유성우처럼 완전 랜덤한 패턴으로 생성되는 모드입니다.
이 때 맵에 길이 있게 하기 위해서 맵을 생성한 후 Floyd-Warshall Algorithm으로 길이 있는지 확인해 주었습니다.
메테오에는 Meteor의 o자가 공으로 바뀌는 이스터에그가 있으니 한번 찾아봐주세요!



파일에는 data와 src가 있는데, 각각 openFramework 프로젝트에서 bin의 data, 그리고 src에 해당합니다. 프로젝트를 새로 생성하신 후 내용을 복사 붙여넣기해주시면 됩니다.
만약 정상적으로 실행이 안 된다면 솔루션 버전 문제일 수 있으니 프로젝트에서 솔루션 대상을 바꿔주시면 됩니다.
